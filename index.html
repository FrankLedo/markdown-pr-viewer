Markdown PR Viewer

Review GitHub pull requests with rendered markdown and inline comments — no VS Code required.

  1. Enter a GitHub repo above (e.g. owner/repo)
  2. Connect with a GitHub token or sign in with GitHub
  3. Pick an open PR to review

Need a token? Create a fine-grained PAT with read access to Contents and Pull requests on the target repo. The token is stored only in your browser's localStorage and never sent to any server.
